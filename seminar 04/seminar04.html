The “you try” question

You try: Draw a different gene at random. Use the t test, with and without the common variance assumption, the Wilcoxon, and/or the Kolmogorov-Smirnov test to assess differential expression. Pull test statistics and/or p-values from the different approaches into an common object, like a readable table. Hint: wilcox.test(), ks.test().

> set.seed(540)

+ {
+   usr <- par("usr"); on.exit(par(usr))
+   par(usr = c(0, 1, 0, 1))
+   r <- abs(cor(x, y))
